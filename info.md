## Navimow Card

Steuerungskarte für den Navimow Mähroboter – angelehnt an die Vacuum-Karte, ohne Karte.

**Funktionen:**
- Status-Anzeige (Mäht / Pausiert / Kehrt zurück / Geparkt / Fehler)
- Schaltflächen: Start · Pause · Zur Ladestation
- Batteriestand mit farbigem Balken

**Minimale Konfiguration:**

```yaml
type: custom:navimow-card
entity: lawn_mower.navimow
```
